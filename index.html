📚 Fundamentos Estadísticos y Funcionamiento del Sistema

El Sistema de Control Estadístico del Hogar es una aplicación web desarrollada para registrar, organizar y analizar los diferentes consumos y gastos que se producen dentro de un hogar.

El sistema permite almacenar información relacionada con servicios básicos como luz, agua e internet, además de gastos personales asociados a alimentación, transporte, estudios, ropa y entretenimiento.

Mediante el uso de técnicas estadísticas descriptivas, el sistema transforma los datos registrados en información útil para ayudar al usuario a comprender sus hábitos de consumo y tomar mejores decisiones financieras.

🎯 Objetivos del Sistema

  • Registrar consumos y gastos del hogar.
  • Mantener un historial organizado de registros.
  • Calcular estadísticas automáticamente.
  • Visualizar información mediante gráficas.
  • Ayudar al control económico familiar.
  • Identificar patrones y tendencias de consumo.

⚙️ Funcionamiento General

  1. El usuario ingresa la fecha, tipo de consumo y valor.
  2. El sistema almacena los datos dentro del arreglo registros.
  3. Se genera automáticamente una observación según el comportamiento del gasto.
  4. Los datos son mostrados en el historial.
  5. Se calculan estadísticas descriptivas.
  6. Los resultados se muestran mediante tarjetas y gráficas.

💻 Tecnologías Utilizadas

  • HTML: estructura del sistema.
  • CSS: diseño visual y responsive.
  • JavaScript: lógica del sistema y cálculos estadísticos.
  • Chart.js: generación de gráficas dinámicas.

📊 Fundamentos Estadísticos

La estadística descriptiva permite resumir y analizar conjuntos de datos mediante indicadores numéricos que facilitan la interpretación de la información.

1️⃣ Media (Promedio)

La media representa el promedio de todos los valores registrados. Se obtiene sumando todos los datos y dividiendo el resultado para la cantidad total de registros.

Fórmula:

Media = Σx / n

Donde:

  • Σx = suma de todos los valores.
  • n = cantidad de datos.
                let suma = valores.reduce(
                    (acc, num) => acc + num,
                    0
                );

                let media =
                suma / valores.length;

2️⃣ Mediana

La mediana es el valor que ocupa la posición central después de ordenar los datos de menor a mayor.

Si existe una cantidad par de datos, se promedian los dos valores centrales. 

                let ordenados =
                [...valores].sort(
                    (a,b) => a-b
                );

                let mitad =
                Math.floor(
                    ordenados.length / 2
                );

La mediana ayuda a conocer el valor central sin verse tan afectada por valores extremadamente altos o bajos.

3️⃣ Moda

La moda es el valor que aparece con mayor frecuencia dentro del conjunto de datos.

Permite identificar el gasto más repetitivo. 

                            let contador = {};

                            valores.forEach(valor => {

                                contador[valor] =
                                (contador[valor] || 0) + 1;

                            });

4️⃣ Valor Máximo

Corresponde al dato más grande registrado. 

                            let maximo =
                            Math.max(...valores);

Permite identificar el consumo más alto registrado.

5️⃣ Valor Mínimo

Corresponde al dato más pequeño registrado. 

                            let minimo =
                            Math.min(...valores);

Permite identificar el consumo más bajo registrado.

6️⃣ Rango

El rango mide la diferencia existente entre el valor máximo y el valor mínimo.

Fórmula:

Rango = Máximo − Mínimo 

                let rango =
                maximo - minimo;

Cuanto mayor sea el rango, mayor será la variación de los datos.

7️⃣ Varianza

La varianza mide la dispersión de los datos respecto a la media.

Permite conocer qué tan alejados están los valores entre sí. 

                            let sumaCuadrados =
                            valores.reduce(

                                (acc, valor) =>

                                    acc +

                                    Math.pow(
                                        valor - media,
                                        2
                                    ),

                                0

                            );

                            let varianza =
                            sumaCuadrados /
                            valores.length;

Una varianza pequeña indica que los datos son similares. Una varianza grande indica una mayor dispersión.

8️⃣ Desviación Estándar

La desviación estándar es la raíz cuadrada de la varianza.

Mide cuánto se alejan los datos del promedio. 

                            let desviacion = Math.sqrt(varianza);

Si la desviación estándar es pequeña, los gastos son relativamente estables.

Si es grande, existen diferencias significativas entre los consumos registrados.

🧠 Aplicación de las Estadísticas en el Proyecto

  • Media: conocer el gasto promedio del hogar.
  • Mediana: encontrar el valor central de los consumos.
  • Moda: identificar gastos frecuentes.
  • Máximo: detectar consumos elevados.
  • Mínimo: identificar consumos bajos.
  • Rango: medir la amplitud de los gastos.
  • Varianza: analizar la dispersión de los datos.
  • Desviación estándar: evaluar la estabilidad económica del hogar.
